주기 기반 고정비 구현 계획

For agentic workers: REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development or superpowers:executing-plans to implement this plan task-by-task.

Goal: 기존 "고정비 항목" UX는 유지하면서 각 항목에 개월 단위 주기를 추가하고, 요약과 차트는 월 환산액 기준으로 계산한다.

Architecture: 계산 규칙은 app/lib/budget.ts에 집중시킨다. UI, CSV, LCM 백업은 이 모델을 소비하도록 얇게 확장한다.

Tech Stack: Next.js, React, TypeScript, Vitest, 로컬 스토리지 기반 상태 저장.

파일 구조

  • app/lib/budget.ts: periodMonths 타입, 정규화, 월 환산 헬퍼, 요약/카테고리 집계 변경.
  • app/lib/budgetImportExport.ts: CSV 템플릿에 주기 컬럼 추가 및 import 파싱.
  • app/lib/backup.ts: .lcm 백업에 periodMonths 저장, 구버전 백업 호환.
  • app/page.tsx: 테이블에 주기 입력과 월 환산 표시 추가, 요약/필터 라벨 조정.
  • app/globals.css: 테이블 컬럼 증가에 맞춘 grid/반응형 스타일 조정.
  • tests/smoke.test.ts: 월 환산 계산, CSV/백업 호환, UI 라벨 영향 테스트 보강.

  const summary = buildBudgetSummary(fixedCosts, 3_000_000);
